# Lógica de Programación  
Curso con uso de Logicode (Pseudocódigo preparatorio para Java)

## Descripción

En esta asignatura trabajaremos el análisis y diseño de algoritmos usando la plataforma **logicode.com.co**, mediante pseudocódigo estructurado como preparación para Java.

El objetivo es desarrollar pensamiento lógico, capacidad de análisis de problemas y correcta construcción de soluciones algorítmicas antes de programar en un lenguaje de alto nivel.

---

## Plataforma de trabajo

- Plataforma: https://logicode.com.co  
- Lenguaje: Pseudocódigo (sintaxis Logicode)  
- Evaluación: Casos de prueba automáticos  

---

## Forma de trabajo

Para cada ejercicio se recomienda seguir el siguiente proceso:

1. Leer completamente el enunciado.
2. Identificar:
   - Entradas
   - Salidas
   - Reglas del problema
3. Diseñar el algoritmo en pseudocódigo.
4. Implementarlo en Logicode.
5. Verificar que la salida coincida exactamente con los casos de prueba.

---

## Reglas importantes en Logicode

- La entrada se realiza con `lea`.
- Se debe respetar estrictamente el orden de los casos de prueba.
- No se deben imprimir mensajes adicionales como: imprima("Ingrese un número").

  - La salida debe coincidir exactamente:
- Orden correcto
- Decimales correctos (usar `formatear`)
- Sin espacios adicionales
- Sin líneas en blanco extra

Ejemplo de formateo correcto: imprima(formatear(resultado,2))
